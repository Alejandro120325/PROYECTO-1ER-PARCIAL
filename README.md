Experiencia web inmersiva de alto impacto visual diseñada para la exploración astronómica. Utiliza un motor 3D fotorrealista y una interfaz futurista bajo la estética *Gold & Dark*.

## 🚀 Características
- **Scattering Cinemático:** Los planetas inician en posiciones orbitales aleatorias para una vista dinámica.
- **Escalado AAA:** Cuerpos celestes con tamaños aumentados y texturas de alta resolución.
- **HUD Futurista:** Panel de control inspirado en *Solar System Scope* con datos técnicos en español.
- **Auth System UI:** Modales premium de Login y Registro integrados.

## 🛠️ Instalación
1. Clonar el repo.
2. `npm install`
3. Colocar texturas en `public/js/textures/` (sun.jpg, earth.jpg, etc.).
4. `npm run dev`
